# ldoc.vim

Generation of ldoc comments for vim.

## Installation
copy `ldoc.vim` to `~/.vim/plugin/`

## Usage
There are 3 types of docs with corresponding commands:
* `:LdocM` - Module
* `:LdocF` - Function
* `:LdocT` - Type
